## 📖 Slash Commands

⚠️ **Note**: Only users with the **`社群管理員`** (Community Manager), **`團長`** (Leader), or **`管理員`** (Admin) roles can operate these commands.

*   **`/record`**
    *   **Function**: Start recording the current channel's chat, or perform a batch export.
    *   **Parameters (Optional)**:
        *   `after_message_id`: Specify a message ID, the bot will start fetching **after** this message.
        *   `before_message_id`: Specify a message ID, the bot will fetch **before** this message (enters batch export mode).
        *   `start_time`: Format `YYYY-MM-DD HH:MM`, fetch messages after this time.
        *   `end_time`: Format `YYYY-MM-DD HH:MM`, fetch messages until this time (enters batch export mode).
        *   `minutes`: Backtrack chat logs for the past specified minutes.
        *   `limit`: Number of past messages to backtrack (up to 100).
        *   `summary`: (Default is True) Whether to generate an AI summary when recording ends.
    *   **Mode Details**:
        *   **Normal Recording**: When no end point (`before_message_id` or `end_time`) is specified, the bot will continue to listen for new messages.
        *   **Batch Export**: When an end point is specified, it grasps messages within the range directly and outputs the file immediately.

*   **`/summary`**
    *   **Function**: Directly generate an AI summary for discussions within the specified range without outputting the full chat log file.
    *   **Parameters**: Same as `/record` (but does not support continuous recording).
    *   **Usage**: Useful when you just want a quick catch-up on discussion highlights from the past and don't need a detailed log file.

*   **`/stop [target_channel]`**
    *   **Function**: Stop recording, and output the chat log Markdown file along with an AI summary.
    *   **Usage**:
        *   Type `/stop` directly: Output files in the current channel.
        *   Type `/stop` and select `target_channel`: Specify a specific channel to send the files to.

*   **`/say [message]`**
    *   **Function**: Send a specific message through the bot.
    *   **Features**: Hides the trace of the command caller, speaking directly as the bot.

## 🛠️ Setup & Installation

1.  **Set up Token**:
    *   Open the `.env` file and fill in your Discord Bot Token: `DISCORD_TOKEN=Your Token`
2.  **Start the bot**:
    *   Run the command in your terminal: `.\venv\Scripts\python main.py`
3.  **Role Permissions**:
    *   Please make sure you have either the `社群管理員`, `團長`, or `管理員` role in your Discord server, otherwise, you won't be able to use the commands.
