# 攔藍錄 (LanLanLu)

[English](README.md) | [臺灣正體中文](README-zh_TW.md)

**「你的每一句話我全都要！！🍔」**

來自蘭蘭露宇宙的⸺**攔藍錄**，是個專門綁架對話紀錄的瘋狂機器人！

> ⚠️ **注意事項**  
> 本專案使用 **Gemini 氛圍編碼** 撰寫，充滿了各種 AI 產生的魔法與不確定性，**使用請注意**！

（若閒置超過 30 分鐘，系統將自動停止錄製以節省資源。）

---

## 📖 指令列表 (Slash Commands)

⚠️ **注意**：僅限擁有 **`社群管理員`**、**`團長`** 或 **`管理員`** 身分組的使用者操作。

*   **`/record`**
    *   **功能**：開始錄製目前頻道的對話內容，或進行批次匯出。
    *   **參數（選填）**：
        *   `after_message_id`：指定訊息 ID，機器人將從該訊息**之後**開始抓取。
        *   `before_message_id`：指定訊息 ID，機器人將抓取至該訊息**之前**（進入批次匯出模式）。
        *   `start_time`：格式 `YYYY-MM-DD HH:MM`，從該時間點後開始抓取。
        *   `end_time`：格式 `YYYY-MM-DD HH:MM`，抓取至該時間點（進入批次匯出模式）。
        *   `minutes`：回溯過往幾分鐘的紀錄。
        *   `limit`：回溯紀錄過去的訊息則數（最大 100 則）。
        *   `summary`：（預設 True）錄製結束時是否產生 AI 摘要。
    *   **模式說明**：
        *   **一般錄製**：未指定「結束點」（`before_message_id` 或 `end_time`）時，機器人會持續監聽新訊息。
        *   **批次匯出**：有指定「結束點」時，將直接抓取範圍內訊息並結案輸出。

*   **`/summary`**
    *   **功能**：直接針對指定範圍的對話產生 AI 摘要，不輸出完整的紀錄檔案。
    *   **參數**：同 `/record`（但不支援持續錄製監聽）。
    *   **用途**：當你只想快速了解過去一段時間的討論重點，而不需要詳細紀錄檔時使用。

*   **`/stop [target_channel]`**
    *   **功能**：停止錄製，並輸出對話紀錄 Markdown 檔案與 AI 摘要。
    *   **用法**：
        *   直接輸入 `/stop`：在目前頻道輸出檔案。
        *   輸入 `/stop` 後選擇 `target_channel`：指定將檔案傳送至特定頻道。

*   **`/say [message]`**
    *   **功能**：透過機器人傳送指定訊息。
    *   **特性**：隱藏指令呼叫者的痕跡，以機器人身分發言。

## 🛠️ 安裝與啟動 (Setup)

1.  **設定 Token**：
    *   開啟 `.env` 檔案，填入您的 Discord Bot Token：`DISCORD_TOKEN=您的 Token`
2.  **啟動機器人**：
    *   在終端機執行指令：`.\venv\Scripts\python main.py`
3.  **權限設定**：
    *   請確保您在 Discord 伺服器中擁有 `社群管理員`、`團長` 或 `管理員` 其中之一的身分組，否則將無法使用指令。
